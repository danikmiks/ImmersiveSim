# ImmersiveSim
ImmersiveSim settled in cyberpunk realities.


__Vector3.hpp__ – pentru poziții și direcții în spațiu.

__Transform.hpp__ – poziție, rotație și scalare.

__Player.hpp__ – informații despre jucător.x	

__Weapon.hpp__ – structura armelor.

__Projectile.hpp__ – gloanțe sau rachete.

__Enemy.hpp__ – inamici de bază.
